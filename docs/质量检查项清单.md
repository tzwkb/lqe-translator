## 二、AI 评估项（17 子类 × MQM/ISO 5060 七维）

评估元规则（每条错误先过）：
1. **三点法则**（客户 LQA 模板）：准确 + 合规（SG/指令/术语）+ 合语法 → 视为正确，**偏好性改写不计错**（至多 Neutral 建议）。防误报第一道闸。
2. **单一归属 + 存疑取重**：按 SKILL.md 归类决策表取最具体一类；Minor/Major 拿不准取 Major。
3. **Word Choice 边界**（客户模板明文）：意思错→Mistranslation；意思对但有更优词→Unidiomatic（Min）或 Neutral。
4. **严重度三档锚**（客户 Error Definition）：Minor=可察觉不误导；Major=很可能误导玩家/损公信；Critical=投诉/法务/功能破坏/高曝光面。

### Accuracy

**Mistranslation 错译 ×1.5**　0512: 2/22+8rep/41+5rep——最大失分源
- 专名↔通名双向误判：拼音残留（画卯→Roll Call，×Mark Mao；平安→Peace，×Ping'an）；通名实体化幻觉（年年≠人名 Niannian；岁月长安≠Chang'an；紫色团花=purple flower，×Epic Flower）
- 玩法规则文本逐条核：数值/上限/门槛（铜筹折算上限 2000）、阶段流程序（海选→小组→半决→决）、机制词（多轮循环=Round-robin，×Rotating）、奖励归属与返还条件——此类错误客户一律 **Critical**
- 操作指引动作语义：技能效果的对象与方式（复原巨石使其停下 ≠ restore…to a halt；招式=any attack，×a Move；"停溯飞行轨迹"被译反）
- 价格/概率语义词不可丢：折扣价、有机会获得（客户 Cri 例：礼盒"contains [68 Gold]" vs 源"有机会获得"）；回復≠receive
- 升 Critical 信号：误导玩家决策的规则文 / 题目-答案失配 / 占位符语义错位 / 经济敏感内容

**Omission 漏译 / Addition 多译 ×1.5**　0512: 0（客户梯度照录备查）
- 限定词清点：任意/1次/仅/同时/最高/每（0512 近失："a Sword Trial" 丢"任意"，被归 Inconsistency）
- Omission 梯度：丢衔接词=Min；丢改变语义成分（"范围内的敌人"→all enemies）=Maj；丢进度/资格关键信息（充值条件）=Cri
- Addition 梯度：贴源过译=Min；偏离源=Maj；引入不存在的玩法元素（单体伤害→all enemies）=Cri

**Untranslated 未翻译 ×1.5 始终Maj**　0512: 0（pre-check 拦截）
- 拼音输出≠已翻译——按 Mistranslation Cri 处理（N1）
- 英文源原样复制也算未译（不止中文残留）

### Terminology ×1.5 始终Maj　0512: 12+8rep——第二失分源
- 活动/玩法/界面入口名逐字对官方库与已上线译名：皇宫寻宝=Imperial Palace Treasure Hunt（自创 Treasures of the Imperial Palace=Maj）；御前练兵=Imperial Drill
- 成对称号体系成对取：新锐|新兵=Recruit / 老将|老兵=Veteran（×New Edge / Old General）
- 人名代号查角色档：青=Halcyon（×Qing 直拼）
- 同物跨段同名：手札统一 Journal（鹤的手札=Crane's Journal，×Note）
- **过度术语化=错**：通名勿造专名（异色灵蝶=strangely colored butterflies，×Spectral Butterfly）
- 系列名格式：赋神·乘桴归梦=Fu Shen - Rippling Dream（·→" - "，系列前缀保留）
- 强制定译清单见 projects/wwm/adjudications.md《0512 裁决》（太极=Cosmic Reversal ×Tai Chi 等 19 条）

### Fluency

**Inconsistency ×1.5**　0512: 5+2rep
- 平行句族统一句型：任务/成就列表同模板（"Complete any X once with a Veteran/Recruit"族，×Team up with… and complete…混用）
- 任务名引用格式统一（Lost Chapter quest: X）
- 涉术语表词条的冲突归 Terminology，其余归此

**Grammar ×1.5**　0512: 0　主谓一致/时态/冠词；占位符邻接可数名词用 "{} day(s)" 型复数

**Punctuation ×1.0**　0512: 0（pre-check 盖大半）　客户梯度：可懂=Min；改句意（Let's eat, Timmy→Let's eat Timmy）=Maj；金额标点=Cri

**Spelling ×1.0**　0512: 0　Equipments/Acheive/Entrence 型；accept/except、then/than、your/you're 混淆=Maj

### Style

**Company style ×1.5**　0512: 12+1Maj
- 句中通名小写：accept a quest / the treasure（×句中 Accept a Quest）——Title Mode 仅限标题位，说明文按句子式
- 序号/卷号 Unicode 罗马数字（N3）
- 括号体系：设施/可放置物名 [Stove]（×「」原样、×#Y"X"#E 引号式）
- 诗句/banner 有押韵节奏重写权（九天阊阖联→bold/gold 韵对）；过度直译判此类
- 禁古英语/网络俚语/UI 语气进剧情（SG 明文违反才归此，无明文归 Unidiomatic）

**Unidiomatic ×1.5**　0512: 6+11rep
- UI 短句自然化模板：`Continue Completing <X>` → `Continue completing the X quest`（去尖括号 + 补通名 quest + 句子化大小写）——0512 同一模式重复 12 次
- 直译腔标志：Warm Tips / Successfully Claimed 型
- 三档标尺（客户 Naturalness）：母语者难懂 / 直译但可懂 / 好文笔——仅前两档计错，第三档不动

### Locale convention ×1.0　0512: 0
- 日期防歧义拼写月份（6/5/2023→May 6, 2023）；货币符号与币种一致（¥299 写成 $299=Cri）

### Audience Appropriateness

**Culture specific reference ×1.5**　0512: 0　520/圣诞吃苹果型文化错置；冒犯/涉政=Cri（客户父类名 Verity）
**Audience appropriateness ×1.5**　本 skill 独有子类　语域/世界观口吻（仙侠敬语→现代俚语）；准确但不合受众期待

### Design & Markup

**Markup ×1.5 / Length ×1.0 始终Maj**　0512: 0（pre-check 盖）　色标相对位置 / {} 数量与顺序 / \n / max-length
**Other ×1.0**　兜底；先过单一归属表再落此
