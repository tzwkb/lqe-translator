## 二、AI 评估项（17 子类 × MQM/ISO 5060 七维）

子类体系、权重 1.5/1.0、四类强制 Major、严重度 0/1/5/10：全部沿用〔SKILL §LQE错误分类；七父维度对齐=GAP-01/02 整改〕。各子类"严重度梯度"列统一取自〔0512def〕客户定义表。

评估元规则（每条错误先过）：
1. **三点法则**：准确 + 合规（SG/指令/术语）+ 合语法 → 视为正确，**偏好性改写不计错**（至多 Neutral 建议）。防误报第一道闸。〔LQA How to Fill，新增〕
2. **单一归属 + 存疑取重**：按归类决策表取最具体一类；Minor/Major 拿不准取 Major。〔SKILL §归类决策规则←调研 §1.4 J2450 元规则，沿用〕
3. **Word Choice 边界**：意思错→Mistranslation；意思对但有更优词→Unidiomatic（Min）或 Neutral。〔LQA 模板明文，新增〕
4. **严重度三档锚**：Minor=可察觉不误导；Major=很可能误导玩家/损公信；Critical=投诉/法务/功能破坏/高曝光面。〔SKILL 严重度判定沿用＋0512def 逐类细化〕

### Accuracy

**Mistranslation 错译 ×1.5**　0512: 2/22+8rep/41+5rep——最大失分源
- 专名↔通名双向误判：拼音残留（画卯→Roll Call，×Mark Mao；平安→Peace，×Ping'an）；通名实体化幻觉（年年≠人名 Niannian；岁月长安≠Chang'an；紫色团花=purple flower，×Epic Flower）〔0512〕
- 玩法规则文本逐条核：数值/上限/门槛（铜筹折算上限 2000）、阶段流程序、机制词（多轮循环=Round-robin，×Rotating）、奖励归属与返还条件——此类错误客户一律 **Critical**〔0512〕
- 操作指引动作语义：技能效果的对象与方式（复原巨石使其停下 ≠ restore…to a halt；招式=any attack，×a Move；"停溯飞行轨迹"被译反）〔0512〕
- 价格/概率语义词不可丢：折扣价、有机会获得（Cri 例：礼盒"contains [68 Gold]" vs 源"有机会获得"）；回復≠receive〔0512+0512def〕
- 升 Critical 信号：误导玩家决策的规则文 / 题目-答案失配 / 占位符语义错位 / 经济敏感内容〔0512〕

**Omission 漏译 / Addition 多译 ×1.5**　0512: 0
- 限定词清点：任意/1次/仅/同时/最高/每（0512 近失："a Sword Trial" 丢"任意"，被归 Inconsistency）〔0512〕
- Omission 梯度：丢衔接词=Min；丢改变语义成分（"范围内的敌人"→all enemies）=Maj；丢进度/资格关键信息（充值条件）=Cri〔0512def〕
- Addition 梯度：贴源过译=Min；偏离源=Maj；引入不存在的玩法元素（单体伤害→all enemies）=Cri〔0512def〕

**Untranslated 未翻译 ×1.5 始终Maj**　0512: 0（pre-check 拦截）
- 拼音输出≠已翻译——按 Mistranslation Cri 处理（N1）〔0512〕
- 英文源原样复制也算未译（不止中文残留）〔GAP-06"未译扩展"〕

### Terminology ×1.5 始终Maj　0512: 12+8rep——第二失分源
- 活动/玩法/界面入口名逐字对官方库与已上线译名：皇宫寻宝=Imperial Palace Treasure Hunt（自创 Treasures of the Imperial Palace=Maj）；御前练兵=Imperial Drill〔0512〕
- 成对称号体系成对取：新锐|新兵=Recruit / 老将|老兵=Veteran（×New Edge / Old General）〔0512〕
- 人名代号查角色档：青=Halcyon（×Qing 直拼）〔0512〕
- 同物跨段同名：手札统一 Journal（鹤的手札=Crane's Journal，×Note）〔0512〕
- **过度术语化=错**：通名勿造专名（异色灵蝶=strangely colored butterflies，×Spectral Butterfly）〔0512〕
- 系列名格式：赋神·乘桴归梦=Fu Shen - Rippling Dream（系列前缀保留〔0512〕；·→" - "=SG 既有规则〔SG〕）
- 强制定译 19 条见 projects/wwm/adjudications.md《0512 裁决》〔0512〕；泛词命中按语境甄别〔ADJ 沿用〕
