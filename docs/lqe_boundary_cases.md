# LQE 边界情况处理决策表（已裁决 2026-06-15 · PM XLY）

目的：把 LQE 处理中所有术语匹配 + 重复/聚合 + 术语库质量的边界情形系统列出，逐条定方案，固化进标准流程。
范围口径（PM 0615 终裁，**已扩大**）：不一致/错误**一律列出**——含 ①不在 TB 的术语各段译法不一致(→Inconsistency) ②TB 占位符/垃圾值 ③TB 自身错误。仅「译员照 TB 填对 + AI 认可」的不报（如 优优，库自身不一致属客户维护）。

例子取自 0615（Batch5+李天明）。**裁决结果**：命名=音译优先；severity 表面拼写=Minor；M7/M9 缺词与父集冲突=走人工门；D2 去重 / D4 同源异译报 Inconsistency / D5 分歧人工核=确认；逐条术语定稿见 `terms_th.json` status=Approved 与 `adjudications.md`。
