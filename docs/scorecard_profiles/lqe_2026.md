# LQE 2026 Scorecard Profile

This profile captures the customer-provided `LQE Report_2026` trial scoring standard.

It is intentionally separate from project profiles:

- Project profiles decide project background, style guides, terminology, checks, adjudications, and wordcount basis.
- This scorecard profile decides severity points, category weights, parent-category labels, aliases, and the customer scorecard template layout.

Use it explicitly with:

```bash
python scripts/lqe_calc.py \
  --state jobs/foo/state.json \
  --errors jobs/foo/errors.json \
  --scorecard-profile lqe_2026
```

The bundled `template.xlsx` is the original customer workbook copied into this profile. Current report generation still uses the internal workbook writer; the template metadata here is ready for a future template-fill exporter.

## Key Differences From Legacy

- `Minor` severity is 2 points instead of 1.
- `Unidiomatic` weight is 3.5.
- `Culture specific reference` weight is 2.5.
- `Terminology`, `Locale convention`, `Inconsistency`, and `Company style` weight is 2.0.
- `Grammar`, `Punctuation`, `Spelling`, `Length`, and `Other` are lighter than legacy.
- `Audience appropriateness` is not a native template subcategory, so it aliases to `Unidiomatic`.
