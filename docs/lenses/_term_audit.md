# 术语自审 · lens 附录

仅术语表/glossary 自审（目标语译文本身是审校对象、无外部 TB 可比，如 Master TB 自审）注入。随各 lens 一并喂入，**覆盖 `_common.md` 的句子流默认**。

## 输入
- 段 = **孤立术语词条**（非句子）。chunk 每条除 `source(中文)`/`target(译文)` 外，附 `en`（英文译名）、`category`、`gender`、`definition`，供 N/A 消歧。
- `kind` = name/desc，由 `category` 推。

## 模式覆盖（覆盖 _common 地盘默认）
- **A 在术语模式只判 desc 段的语义**；**name 段的专名音译归 N**（避免一个 agent 既判音译又判词义而锚定漏挖）。
- 门控：N 只跑 name；A、R 只跑 desc；T、G 跑全部。
- **外部 TB 比对关闭**：TB 自身是审校对象，自审时 `term_hits`/`term_near` 为空，T 转**纯内部一致性 + 占位/垃圾值**甄别（不拿外部 TB 对照）。

## 共同提醒
- **占位/垃圾值**：译文用通用词顶替专名（义为「动物/物品」之类），或误填了他条目的值 → A 报（语义未译/误填）；跨词条层面 T 汇总。
- **系列/家族上下文**：同进化链/系列的词条命名应成体系、组件译法对齐（N 看专名族、A 看描述族）。
- 逐条过，**别因词短跳过**——原子短词同样会词义漂移 / 音译走形。
