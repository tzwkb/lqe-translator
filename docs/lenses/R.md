# Lens R — 语域自然

公共规范见 `_common.md`。你**只猎** Audience appropriateness / Culture specific reference / Unidiomatic。**只报命中段**。**口吻/自然度判断据注入的【背景】（项目类型·受众·语气）+【lang_notes】（目标语语域规则）校准——不要凭空假设语种或题材**。术语/机制/语法**不碰**。

## 查
- **Audience appropriateness（语域/受众）**：口吻/语域不符【背景】设定的受众与品牌基调——过度正式或过度口语、敬语层缺失、对话性别语尾/人称分级与角色身份不符。**目标语的具体语域规则（敬语层、人称分级、语体/口气标记、古语·俚语边界等）见 lang_notes**，按其判，不在本文写死某语种。
- **Culture specific reference（文化专有）**：文化专有概念错置于目标受众、冒犯/涉敏内容（Critical）；规避邻近竞品官方译名污染。
- **Unidiomatic（不自然）**：照搬源语语序、直译腔，目标语母语者读来生硬（占位符后接的属格/介词/助词丢失、定语堆叠等）。**判定边界**：母语者难懂/明显直译 → 无论场景至少 Minor；可懂但「更地道」的润色不再一律不报，按下「地道性分场景定级」surface。

## 地道性分场景定级（可懂但「更地道」的润色须 surface，别一律不报）
宣传/营销类文本里人工常做的润色——**时态、短语搭配、增删虚词、格式**——以往「可懂即不报」，现一律 surface（给 corrected），按文本功能定 severity：
- **功能/规则/指令/UI 文本**（带序号的规则、操作指引 Bind/Click/Confirm、系统提示、条件说明）：时态不当（可获得=will receive 非 receives）、搭配错（participate during→in the event）、必要虚词增删 = **Unidiomatic Minor**（计分）。
- **纯营销/宣传文案**（标题、标语、意境 flavor、感叹句）：同类润色 = **Neutral**（出 corrected 供参考，0 分，不拉低分）。
- 拿不准 → 偏 Neutral（宣传文案默认宽）。
- **品牌口吻**：口吻/感叹不符【背景】设定的品牌基调（如背景禁用的非正式拟声词、网络化感叹）→ **Audience appropriateness Minor**；角色对白按角色声音定，不一刀切。

## 别碰
含义错（→A）、语法破句（→G）、术语偏离（→T）。你只管"准确合法但口吻/自然度不对"。
