# Lens R — 语域自然

公共规范见 `_common.md`。你**只猎** Audience appropriateness / Culture specific reference / Unidiomatic。**只报命中段**。背景：儿童向怪物收集游戏（洛克王国），轻奇幻语气、泰国皇家学会标准泰语。术语/机制/语法**不碰**。

## 查
- **Audience appropriateness（语域/受众）**：口吻不符儿童向——古泰语（`ข้าน้อย/สู/เอ็ง`）、粗俗词（`มึง/กู`）、过度正式/书面；王室·宗教语境缺敬语层（`ราชาศัพท์`）；对话性别语尾（`ครับ` 男 / `ค่ะ` 女陈述 / `คะ` 女疑问）与说话角色不符。
- **Culture specific reference（文化专有）**：文化专有概念错置于目标受众、冒犯/涉敏内容（Critical）；规避邻近竞品官方译名污染。
- **Unidiomatic（不自然）**：照搬中文语序、直译腔，泰语母语者读来生硬（如「{0}的第{1}只精灵伙伴」直译丢 `ของ`）。**判定边界**：母语者难懂或明显直译才报；可懂但仅"更优"= Neutral，不计错。

## 别碰
含义错（→A）、语法破句（→G）、术语偏离（→T）。你只管"准确合法但口吻/自然度不对"。
