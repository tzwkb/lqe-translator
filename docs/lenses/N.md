# Lens N — 专名音译

公共规范见 `_common.md`。术语自审专用 lens（句子流不派此 lens）。你**只猎专名音译错**，归 **Mistranslation**（文化负载名→Culture specific reference）。**只报命中段**（无错段不输出）。描述词义归 A、拼写正字归 G、一致性/垃圾值归 T、语域归 R——一律不碰。

## 覆盖
只跑 `kind=name` 段（角色/怪物/地名等专名）。chunk 每条带 `en`（英文译名=音译锚）、`category`、`definition`、`gender` 供消歧。

## 判法
专名的目标译应与**目标读音一致，以 `en` 为罗马化锚**；`en` 为空时据**中文源读音**判。报错若：
- 整名认不出 / 读音明显不符锚（Major）
- 丢音节 / 增音节 / 首辅音换掉（两音节名缩成单音节、Trinity 译成另一个名）
- 撞真实品牌或他名（音译读出已知商标/无关名）
- 人名↔地名 / 类别张冠李戴（人名安成某地名、套用了另一条目的译名）
- 违反既定专名 / 禁用名（见 adjudications）
- 目标文字 / 罗马化系统用错、音节结构明显走形

整名认不出 = Major；单个音节/元音小误、整体仍可辨 = Minor。纯字符/声调符号层面的拼写错交 G（除非整名因此认不出）。文化负载名（典故/神祇/特殊含义名）译错 → Culture specific reference。

## 中立（靠注入，勿硬编）
- 罗马化/转写惯例、声调与元音在音节中的位置规范 → 从 `lang_notes`
- 既定专名表、统一译名、禁用名、音译优先与否 → 从 `adjudications`
- 题材（决定专名风格走向）→ 从 `background`

本文件不写任何具体语种或具体游戏的译名。

## 不要报
- 与他段译法不一致（归 T）；纯目标文字拼写/声调符号错（归 G，除非整名因此认不出）；描述性短语的词义（归 A）。
- 可接受的风格变体、有意保留的拉丁品牌名。
- ZHCN 与 EN 冲突时，专名以 **EN 锚读音**做音译；若 EN 本身明显错置（与中文完全无关的他名）→ 标交人工，勿硬猜。
