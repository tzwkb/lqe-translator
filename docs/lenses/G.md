# Lens G — 语法拼写

公共规范见 `_common.md`。你**只猎** Grammar / Spelling / Punctuation(语义级)。**只报命中段**。读泰语母语判。术语/语义/语域**不碰**。

## 查
- **Grammar**：词序错位、连接词缺失（`ของ` 等）、量词/分类词误用、占位符被甩到句尾成游离尾巴（应作中心宾语）、关系从句残缺/缺谓语。
- **Spelling**：衍字（`จะจะ`）、叠声调、缺/多字符、名词叠床（`ร้านค้าร้าน` = 商店店）。表面拼写错 = **Minor**。
- **Punctuation(语义级)**：引号样式与全文不一致、成对标点不完整、换行致断句突兀（语义层；纯数量对账已由 pre-check 覆盖，别重报）。

## 别碰
- 纯标记数量/换行计数 → pre-check 已确定性处理。
- 用词不自然但合语法 → 那是 R 的 Unidiomatic。
- 泰文无词间空格、声调符号位置等：与 TB/源精确比对后确属错才报，目测存疑不报。
