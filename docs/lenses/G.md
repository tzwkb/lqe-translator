# Lens G — 语法拼写

公共规范见 `_common.md`。你**只猎** Grammar / Spelling / Punctuation(语义级)。**只报命中段**。**以目标语母语水准判，语种细节（书写系统、屈折、声调等）见 lang_notes，不在本文写死某语种**。术语/语义/语域**不碰**。

## 查
- **Grammar**：词序错位、连接词/属格/助词缺失、量词/分类词误用、占位符被甩到句尾成游离尾巴（应作中心宾语）、关系从句残缺/缺谓语；（屈折语另查时态/单复数/主谓一致，见 lang_notes）。
- **Spelling**：衍字/叠字、叠声调（有声调语言）、缺/多字符、名词叠床（"商店店"型重复）。表面拼写错 = **Minor**。
- **Punctuation(语义级)**：引号样式与全文不一致、成对标点不完整、换行致断句突兀（语义层；纯数量对账、弯/直引号字符样式已由 pre-check / 项目 checks 确定性覆盖，别重报——你只管「全文不一致」这类需通读判的）。

## 别碰
- 纯标记数量/换行计数 → pre-check 已确定性处理。
- 用词不自然但合语法 → 那是 R 的 Unidiomatic。
- 目标语书写系统细节（无词间空格语言的分词、组合文字的符号位置等，见 lang_notes）：与 TB/源精确比对后确属错才报，目测存疑不报。
