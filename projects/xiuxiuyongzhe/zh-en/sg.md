# 咻咻勇者中英本地化风格指南

来源：`sources/风格指南-咻咻勇者.xlsx` 的可见工作表“内部风格指南”。原表中的 Regex 仅为人工搜索提示；规则判定须结合变量、标签、缩写和上下文。

## 标点与空格

- 半角逗号、冒号、句号、感叹号、问号、分号后加一个半角空格；紧跟其他符号、句末、缩写、代码、变量或标签时按适用例外处理。逗号前不留空格。
- 连字符 `-` 用于复合词，前后不留空格。
- En dash `–` 用于数字范围，前后不留空格。
- Em dash `—` 用于停顿，通常前后不留空格。
- `%` 前不留空格；`+` 与 `$` 后不留空格。
- 半角单双引号外侧按英文句法留空格，内侧不留空格。
- 半角左括号前加空格，括号内部不留空格。
- 英文文本中的中文方括号 `【】` 直接删除，不替换为其他括号。
- 撇号用于所有格或缩写，前后不留空格。

## 省略号

- 英文省略号使用三个半角点 `...`，不用中文六点；句末另需句号时可写四个点。
- 表示句中停顿、中断或拖长时，`...` 前后各留一个空格。
- 表示 UI 加载或未完待续时，`...` 前不留空格，后不接其他字符。

## 数字、单位与标签

- 小数使用点号，数字与小数点之间不留空格。
- 四位及以上数字使用英文千分位逗号，如 `1,000`。
- 阿拉伯数字或数字 Tag 与英文字母、单词之间加半角空格。
- 固定形式如 `3D`、`MP4`、`PS5`，以及 `s` 表示 second 时不加空格。
- `Lv.` 后加半角空格再接数字，如 `Lv. 50`。
- 图标 Tag 与数字、文字之间加空格；与符号之间不加空格。
- 换行符与相邻文本之间不留空格。
- 道具编号沿用源文阿拉伯数字，不改为罗马数字。
- 若源文没有句号，译文不要自行增加句号；需结合文本功能与其他句末标点复核。
