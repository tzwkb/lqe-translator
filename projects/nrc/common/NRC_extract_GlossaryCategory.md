# NRC 术语分类法 — [Glossary Category] tab 全量转录 2026-06-10
角色：术语登记/QA 报告中术语错误的归类标准。列：术语类别 | 子类别 | 例子 | 解释/备注（中英双语原文）。

| Category | Sub Category | 例子 | 解释/备注 |
|---|---|---|---|
| Character Name | Named NPC | 马尔文、路易斯 | 具有固定姓名或专属称谓的角色。Characters with fixed names or unique titles. |
| Character Name | Generic NPC | 兴奋的学生、庄重的卫兵 | 无固定姓名，以身份、职业或状态指代的角色。Characters identified by role, profession, or description rather than a unique name. |
| Character Name | Enemy NPC | 黑巫师、守门黑巫师 | 在玩法层面与玩家对立、可战斗或敌对的角色。Characters hostile to the player in gameplay. |
| Jini | Creature Species | 小独角兽、噜噜 | 精灵的种族或物种名称。备注里的 1/3 2/3 3/3 指同一只精灵的进化形态，**取名字时建议同一进化链一起考虑**。Species-level names; naming should consider the full evolution chain for consistency. |
| Jini | Creature Individual | 花花、船长噜噜 | 具有独立身份或特定指代的精灵个体。Specific individual creatures. |
| System Narrative Object | Interactive Narrative Object | 皮箱、巨大的石碑 | 可被玩家互动或触发剧情的物件。 |
| System Narrative Object | Narrative Element | 渺远的声音 | 非实体或不可交互的剧情表现元素。 |
| Lore | Core Lore | 精灵、洛克、契约 | 世界观的基础概念与核心设定。 |
| Lore | Faction | 白银骑士团 | 组织、团体或势力单位。 |
| Lore | Title | 大魔法师 | 角色所拥有的身份或荣誉称号。 |
| Gameplay | Gameplay System | 精灵对决、精灵技能 | 核心玩法机制或战斗模式名称。 |
| Gameplay | Key Items | 魔杖、咕噜球 | 在世界观或玩法中具有重要地位的关键道具。 |
| Gameplay | World Resources | 许愿星、绩点 | 可收集、使用或转化的资源类物品。 |
| Gameplay | Overworld Exploration Abilities | 愿力魔法、浮风术、液化术 | 用于地图探索与移动的能力。 |
| Gameplay | Overworld Special Mechanics | 连续捕捉、噩梦领域 | 影响大世界玩法的特殊规则或机制。 |
| Gameplay | Overworld Utility Systems | 魔力之源、炼金营地 | 构成大世界运作的功能组件。 |
| Gameplay | Presentation Elements | 音乐 | 地上可以捡起来的音乐，捡起来了可以播放。 |
| Gameplay | Social Features | 回响之花、留言 | 玩家之间互动的系统或元素。 |
| Cosmetics | Costumes | 噪点裂变、昼梦信使 | 时装品牌！ |
| Cosmetics | Cosmetics system | 皮卡月刊 | 长期存在的核心功能系统（也是时装相关的）。 |
| System | PvE Content | 稀兽花种 | PvE 玩法的延伸内容。 |
| System | PvP Content | 巡回天梯 | PvP 玩法的延伸内容。 |
| System | Seasonal System | 赛季大厅 | 以赛季为周期的运营或玩法系统。 |
| System | Jini Base Stats | 生命、物攻、物防 | 精灵的核心属性数值。 |
| System | Jini Growth System | 种族值、个体值 | 影响精灵成长与养成的机制。 |
| System | Jini Traits System | 特性、血脉、奇异血脉 | 精灵的特性、血脉或被动属性机制。 |
| System | Jini Combat System | 技能 | 精灵参与战斗的技能与战斗机制。 |
| System | Jini Evolution System | 分支进化 | 精灵形态变化或进阶机制。 |
| System | Jini Personality System | 性格 | 精灵的个性或行为特征设定。 |
| System | Jini Rank Tags | 首领 | 用于标识精灵身份或阶级的标签。 |
| Map | Functional Area | 战斗学分院 | 具有明确功能或用途的场所（学院、要塞、设施）。 |
| Map | Administrative Region | 洛克王国、风眠省 | 具有治理或统治结构的正式区域。 |
| Map | Macro Region | 王国海岸南部、独角兽领地、第五象限 | 大尺度地图划分，不强调政治属性。 |
| Map | Urban Area | 彼得大道、商店街 | 城市内部或城镇核心街区。 |
| Map | Settlement | 聆风镇、信仰者村落 | 可居住的小型城镇、村落或港口。 |
| Map | Wilderness | 原野空径、沉船漩涡 | 自然环境区域（山脉、森林、湖泊、荒地）。 |
